Welcome to cicada-notebook ^-^/

  • An interactive UI of cicada-language
  • minimalism by design

    • Buttons

  • RUN -- run all the note before current note
  • NEW -- RUN & create a new note after current note
  • DEL -- delete current note
  • # -- toggle table of content
  • LOAD -- load a .cic file -- //// line will be load as headline
  • SAVE -- save current notebook as a .cic file -- headline will be save as //// line

    • Key Bindings

  • Ctrl + Enter -- RUN
  • Alt + Enter -- NEW
